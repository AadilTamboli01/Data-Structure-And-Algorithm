# DSA Practice Repository

This repository contains my practice of **Data Structures & Algorithms (DSA)** problems.  
All problems are solved in **Java**, and each solution is in a separate file for easy understanding and testing.

I have organized problems by topics like **Arrays, Linked Lists, Stacks, Queues, Recursion, and Dynamic Programming**.  
The goal of this repository is to improve problem-solving skills, learn DSA concepts, and prepare for coding interviews.
